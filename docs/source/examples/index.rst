Examples
========

Examples of typical DYCOVE usage are documented here. 
The examples cover usage both `ANUGA <https://github.com/GeoscienceAustralia/anuga_core>`_ and `Delft3D Flexible Mesh (DFM) <https://oss.deltares.nl/web/delft3dfm>`_ models.

To run each example, you must navigate to the corresponding directory under "examples/".
From there, you may inspect and run the provided Python script.
While the examples are self-contained within their respective directories, the documentation provided here offers more of a step-by-step guide with some additional explanation.
